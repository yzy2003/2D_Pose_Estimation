# 2D_Pose_Estimation
==基于2D人体姿态估计的体态评估分析系统设计与实现 -- 毕设设计==
## 1. 课程目标
> 本课题围绕基于人体姿态估计的体态评估分析展开研究，旨在将计算机视觉与健康评估方法相结合，构建一套集姿态识别、体态分析与可视化展示于一体的智能体态评估系统。通过对人体姿态估计算法的研究与应用，实现对人体关键点的精准提取，并结合人体解剖学和运动学知识，对常见体态问题进行量化分析与评估，为用户提供直观的体态评分与健康建议。同时，借助现代Web开发技术，完成系统的工程化实现，验证该方法在实际应用场景中的可行性与实用价值。
## 2. 课程内容
- **人体姿态估计方法的研究与实现**
深入学习人体姿态估计的基本原理和主流算法，理解从单张图像或视频序列中提取人体关键点的实现过程，并完成至少一种姿态估计算法的本地部署与推理，为体态分析提供可靠的基础数据。
- **体态评估指标设计与分析方法研究**
结合人体解剖学与运动科学知识，围绕头前倾、高低肩、骨盆前倾等常见体态问题，设计基于人体关键点的几何特征与角度指标，建立一套可量化的体态评估方法，实现对用户体态状态的客观分析与判定。
- **后端服务接口设计与实现**
利用FastAPI框架对姿态估计与体态评估算法进行封装，设计高性能、易扩展的后端接口，实现图像或视频数据上传、姿态分析与评估结果返回等功能，支撑系统的整体运行。
- **前端系统开发与可视化展示**
采用React前端框架开发用户交互界面，实现人体关键点可视化叠加、体态评分展示及健康建议输出等功能，提升系统的易用性和交互体验，最终形成前后端协同工作的完整体态评估分析系统。
